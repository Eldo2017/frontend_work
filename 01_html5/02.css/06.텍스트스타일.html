텍스트 스타일(text~)

1. color : 텍스트 색상을 만든다

        선택자 {
            color : 색상명 | 16진수 | rgb(x,x,x) | rgba(x,x,x,x) | hsl(x,x,x) | hsla(x,x,x,x)
        }

2. text-decoration : 텍스트에 줄 긋기나 줄을 제거할 때 사용

        선택자 {
            text-decoration : none | underline | overline | line-throw
        }
  • 텍스트 영역 밑에 줄긋기
  • 텍스트 영역 중간에 줄긋기
  • 텍스트 영역 위에 줄긋기
치지직으로!

3. text-transform : 영문 텍스트의 대소문자 변환에 사용

        선택자 {
            text-transform : uppercase | lowercase | capitalize;
        }
  1. uppercase : 모든 영문자를 대문자로 바꾼다
  2. lowercase : 모든 영문자를 소문자로 바꾼다
  3. capitalize : 모든 영문자 중 맨 처음에 있는 글자를 대문자로 바꾼다

4. text-shadow : 텍스트에 그림자 효과를 준다

        선택자 {
            text-shadow : x(가로) | y(세로) | 번짐정도 | 색상
        }

표 스타일

        선택자 {
            - border-collapse : separate | collapse
              표 테두리와 셀 테두리를 합친다

            - border-spacing : 간격
              셀 사이의 공간을 조정한다
              border-collapse : separate;하고 사용하여야 한다
              (collapse는 변화가 없다)

            - table-layout : auto | fixed
              테이블 크기와 셀 너비는 컨텐츠에 따라 자동으로 결정이 된다(기본값)
              내부의 내용하고 상관없이 테이블의 크기를 고정한다

            - vertical-align : top | middle | bottom
              세로 측의 글씨를 정렬한다
        }

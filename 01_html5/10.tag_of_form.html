폼 관련 태그

form 태그 내 submit 버튼을 클릭 시 해당 폼에 작성이 된 사용자가
입력한 값들이 서버로 전송이 되며, 서버에 요청하는 역할을 한다.
<form action="" method=""> * action 속성 : 해당 form의 사용자가 입력한 값들을 전달하며 요청할 서버의 경로를 제시한다
* method 속성 : 요청 전송 방식을 제시한다(get | post)
>> get 방식 : 요청 시 사용자가 입력한 값들이 url에 노출이 되는 방식(기본값) >> post 방식 : 요청 시 그 값들이 url에 노출이 되지 않는 방식 form 내의 submit 버튼을 클릭하면 aciton에 지정한 서버로 사용자가 입력한 값들이 넘어가면서 요청된다(사용자가 입력한 값들은 항상 key=value 세트로 전달이 된다)

- 노출시키지 않고도 서버로 전송해야 하는 정보가 있을 때 사용한다


type="submit | reset | button"

- submit : 폼의 모든 값들을 서버로 전송시킨다 

- reset : 폼의 모든 값들을 초기화(원래대로 되돌리기)한다 

- button : 일반 버튼 

button[type="submit|reset|button]"
